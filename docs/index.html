GRID: Deal with Field Segmentations Elegantly¶

Core features¶

Getting started¶

  • Installation: Python 3 | Dependencies | PyPI |

  • First-time users: Demo mode | Work with your images |

Interface¶

  • File loading: Inputs | Drag and drop

  • Area of interest: Define AOI | Adjust AOI

  • K-means clustering: Clustering | Binarization | Refine clustering | Display/Zoom | Rotate images

  • Centroids searching: Major axis | Minor axis | Switch between axes | Move/add/delete centroids

  • Segmentation: Dynamic | Fixed | Fine-tune results | Reset | Display | Export results |

Inputs/Outputs¶

  • Inputs: Images | Maps | Shape files

  • Outputs: Tabular results | Images for validations | Shape files | NumPy form of AOI | H5 dataset

Advanced usages¶

  • Images from multiple seasons: Run GRID for the 1st season | Apply shapefiles to another season | Compare different seasons

  • Work with QGIS: Obtain shapefiles from GRID | Load shapefiles to QGIS

  • Customize vegetation indices: Obtain tabular outputs | Derive indices | Visualize indices

  • Adapt to an arbitrary field layout: What’s an arbitrary layout | Define centroids in GRID | Intepretate results

  • Generate datasets for deep learning: Obtain h5 datasets | Process h5 files | Construct a DL model | Train & Evaluate
